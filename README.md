# Bazarigo

🌐 **Live Website:** https://www.bazarigo.com/

Bazarigo is a modern e-commerce and marketplace web application built with React and Vite. It is designed to deliver a fast, scalable, and user-friendly shopping experience with advanced product management, analytics, and interactive UI features.

---

## 📌 Overview

Bazarigo provides a complete digital commerce solution with features like product browsing, cart management, authentication, dashboards, analytics, and rich product interactions. The system is optimized for performance, scalability, and modern UI/UX standards.

---

## 🛠️ Tech Stack

### Core

- React 19
- Vite (Rolldown Vite)
- React Router 7

### Styling & UI

- Tailwind CSS
- DaisyUI
- Framer Motion
- Lucide React Icons
- React Icons

### State & Data Management

- @tanstack/react-query
- Axios
- js-cookie

### Forms & Inputs

- React Hook Form
- React Datepicker
- React Rating

### Charts & Visualization

- Recharts

### Rich Content & Utilities

- Quill Editor
- SweetAlert2
- Moment.js
- UUID
- XLSX (Excel export/import)

### Media & UX Enhancements

- React Responsive Carousel
- React Image Magnify
